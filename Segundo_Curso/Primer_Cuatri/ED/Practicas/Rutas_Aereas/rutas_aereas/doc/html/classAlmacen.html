Detailed Description

Una instancia del T.D.A Almacen representa un conjunto de rutas, diferenciadas por su identificador. De esta forma a cada ruta del almacén le corresponde un único identificador.

◆ Almacen() [2/2]

Almacen::Almacen ( const char  fich_almacen[])

Constructor. Recibe un fichero que contiene la información del almacén. El formato del fichero será el siguiente:

  • En la primera línea aparece la cadena mágica, que identifica al fichero como uno que contiene los datos de un almacén.
  • En las líneas siguientes, aparecen las rutas almacenadas en el almacén.
Parameters
fich_almacenfichero que contiene los datos del almacén.
Exceptions
std::ios_base::failuresi no se pudo abrir el fichero.
throwstd::invalid_argument si no se encuentra la palabra mágica en el fichero.
Postcondition
El almacén contendrá las rutas que aparecían en el fichero y se eliminarán todas las rutas que hubiera almacenadas antes.
